# 傅里叶级数 (Fourier Series)

本模块旨在提供傅里叶级数的核心收敛理论、展开计算流程，以及其在现代信号处理中的工程映射。请根据学习需求选择进入以下板块：

### [1. 阅读材料](reading.md)
* **内容**：傅里叶级数的无穷维正交基本质，及其在声学频谱（音色）、极限数据压缩（偶延拓）和物理系统边界约束（奇延拓）中的实际应用。
* **用途**：用于在掌握基础计算工具后，深刻理解频域分析在现代数字信号处理与物理工程中的底层逻辑。

### [2. 理论速查](review.md)
* **内容**：汇总周期函数展开的狄利克雷条件，以及傅里叶级数在连续点、跳跃间断点和区间端点处的严格收敛定理。
* **用途**：作为核心理论字典，用于在特定点求和题型中，快速核对级数的收敛目标值。

### [3. 拓展阅读 ](material.md)
* **内容**：对傅里叶级数的补充说明以及奇偶延拓的实际意义
