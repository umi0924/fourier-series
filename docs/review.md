# 傅里叶级数：收敛性定理速查

本模块总结了周期函数展开为傅里叶级数的前提条件，以及级数在不同类型坐标点（连续点、间断点、区间端点）上的收敛目标值。

---

## 一、 展开前提：狄利克雷条件 

设 $f(x)$ 是周期为 $2L$ 的周期函数。若 $f(x)$ 在一个周期 $[-L, L]$ 内满足以下三个条件（狄利克雷条件），则其傅里叶级数必定收敛：
1. **连续性限制**：在一个周期内只有有限个第一类间断点（即左右极限均存在的跳跃间断点）。
2. **极值点限制**：在一个周期内只有有限个极值点。
3. **绝对可积**：在一个周期内绝对可积，即 $\int_{-L}^{L} |f(x)| dx < \infty$。

*(注：在绝大多数工程与物理实际问题中，信号函数均天然满足狄利克雷条件。)*

---

## 二、 狄利克雷收敛定理 

当 $f(x)$ 满足狄利克雷条件时，其傅里叶级数 
$$S(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos\frac{n\pi x}{L} + b_n \sin\frac{n\pi x}{L} \right)$$
在整个实数轴上的收敛性遵循以下严格法则：

### 1. 在连续点处的收敛性
* **判定准则**：若 $x_0$ 是函数 $f(x)$ 的连续点。
* **收敛结果**：傅里叶级数严格收敛于该点处的函数值。
  $$S(x_0) = f(x_0)$$

### 2. 在间断点处的收敛性
* **判定准则**：若 $x_0$ 是函数 $f(x)$ 的跳跃间断点。
* **收敛结果**：傅里叶级数收敛于该点**左极限与右极限的算术平均值**。
  $$S(x_0) = \frac{f(x_0^-) + f(x_0^+)}{2}$$
* **工程现象**：在间断点附近截断有限项级数时，会出现无法消除的震荡（吉布斯现象），但当项数趋于无穷大时，该点的级数值被严密地“钉”在跳跃幅度正中央的平均值处。



### 3. 在区间端点处的收敛性 (针对非周期函数的周期延拓)
* **判定准则**：当仅定义在 $[-L, L]$ 上的非周期函数被强行进行周期延拓时，端点 $x = \pm L$ 往往成为新周期函数的间断点。
* **收敛结果**：傅里叶级数在区间端点 $x = -L$ 和 $x = L$ 处，均收敛于两端点函数值的算术平均值。
  $$S(\pm L) = \frac{f(-L^+) + f(L^-)}{2}$$

---

## 三、 收敛性判定速查表

| 目标点 $x_0$ 的类型 | 傅里叶级数 $S(x_0)$ 的收敛值 | 公式表达 |
| :--- | :--- | :--- |
| **内部连续点** | 原始函数值 | $f(x_0)$ |
| **内部跳跃间断点** | 左右极限的中点 | $\frac{1}{2}[f(x_0^-) + f(x_0^+)]$ |
| **区间边界点** $(\pm L)$ | 两边界端点值的平均 | $\frac{1}{2}[f(-L^+) + f(L^-)]$ |

!!! warning "解题警示"
    在求解傅里叶级数的特定点求和题型时（例如利用傅里叶级数求 $\sum \frac{1}{n^2}$ 的值），**第一步必须判断代入的 $x$ 值究竟是连续点还是间断点**。若为间断点而直接令级数等于 $f(x)$，将导致计算结果发生重大错误。
